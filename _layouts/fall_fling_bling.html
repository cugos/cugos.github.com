CUGOS 2013 Fall Fling

☰ Menu

Seattle

Wednesday, October 16, 2013 UW Anderson Hall Register Now!

A conference for open-source geospatial people who need to talk.

Seating is NOT limited — Register Anyway!

The CUGOS Fall Fling is an intensely educational learning session for passionate practitioners of all things open-source geospatial. If you care about geometry busting workflows, academic research brain dumps, parsing vertices while drinking beer, code as well as content, usability as well as design, then show up and eat free pizza. Join us for this special all-day-kinda event at the magical University of Washington Seattle Campus in place of our regular monthly meeting. BOOYA!

  • Location

    The 2013 Fall Fling will be held at UW Anderson Hall. We didn't need to arrange special room rates for attendees because you can't sleep there. Full Details

  • Registration

    There is NO on-site registration or badge pick-up. Just pop over to the Eventbrite page to register so we get a headcount on food. And show up fully clothed on the day of the event.

  • Amenities

    Leave the brown bag at home! Thanks to our sponsors you will be eating a free lunch with free drinks. It's a free event so don't expect too much.

Wednesday, October 16

A Gentle Reminder

The event runs from 9:00am – 4:00pm and features talks, more lightning talks, lunch and how-to sessions. There’s a lot to cover, so show up on time...especially if you are a speaker.

Intro and Keynote

  1. 9:00am-9:05am

    Introduction

    Aaron Racicot, Godfather of CUGOS and MapFart.com, Z-Pulley Inc.

    Thou shalt love GDAL. Thou shalt hug QGIS. Love thy Python scripts as thyself. And more!

  2. 9:05am-9:25am

    Keynote -- Replacing ArcGIS Server with NodeJS and PostGIS

    Ryan Whitley, Software Developer, SpatialDev

    How I’m using Node.js to create a lightweight API on top of PostGIS for Feature Services, GeoProcessing workflows and TopoJSON support. TileStream and TileMill make for easy map caching and a tile server, and a generic map print service that works! Bundled together with glue and javascript, this combination provides a low-cost alternative to ArcGIS Server.

Short Talks

  1. 9:30am-9:50am

    Touring the GINA Zoo -- Big Data, Web Apps, Lions

    Will Fisher, GINA Alaska

    Mr Fisher is coming all the way from Alaska to demonstrate GINA workflows. Yeah!

  2. 9:50am-10:10am

    Big Data Worship -- Aerial Structure from Motion Surveys and Antarctica/Greenland DEMs

    David Shean, Academic, UW Department of Earth and Space Sciences

    Mr Shean does fascinating research and has pretty data to go with it based on his September CUGOS teaser!

  3. 10:10am-10:30am

    Geoserver Shell -- Manage GeoServer using a CLI

    Jared Erickson, Pierce County Developer, Author of GeoScript-Groovy

    Jared builds great tools. Some of the things Jared has built: GeoScript Groovy Bindings, GeoScript Scala Bindings, Geometry Commands

  4. 10:30am-10:50am

    WebGIS for Decision Support -- Problem Solving with OpenSource GIS

    Karsten Vennemann, Geographer, Terra GIS Ltd

    Karsten will give a tale of personal experience about how he has been using open-source components to build web maps that help to solve real-world problems. Examples include invasive weed management & prioritization in California ( Whippet ), a commercial system for automatic web based image processing and map delivery ( GeoSpatial Partners ) , and VFS Viewer - an information system for privately owned forest areas in the State of Bavaria, Germany.

  5. 10:50am-11:10am

    Empower ENGINE

    Julie and Noah, Owners, EmpowerENGINE.com

    Make powerful data visualization accessible to progressives up and down the ballot!

    A description about your talk here.

Lightning Talks

  1. 11:30am-11:40am

    Closure GDS for Geo

    James Keppy Dominguez, Hacker Extraordinaire

    Graph Data System (GDS) with a leveldb persistence layer and real time data aggregation. Written in Clojure. Pumping out GeoStuffs

  2. 11:40am-11:50am

    D3.geo

    Sam Matthews, Founder, Situated Labs

    Why D3 is great and how you can get a map running on your website quickly and efficiently.

  3. 11:50am-12:00pm

    TileStache

    Matt Kenny, GIS Data Engineer Tableau Software

    See how you can use this Python tile server to render TopoJSON tiles and animated cat GIFs.

  4. 12:00pm-12:10pm

    Raster Edge Detection

    Andrew Powers, Biz Ops Data Analyst, Point Inside

    Hear how a few small changes to popular edge detection algorithms give you the power to detect human made features in satellite imagery.

  5. 12:10pm-12:20pm

    Data Visualization of Urban Systems with GeoServer & Google Earth - Decisions for DRiVE

    Luis F. Borrero, DRiVE

    Luis will talk about how DRiVE leverages the Earth platform to provide decision makers with unique 3D-Visuals in the context of urban systems. With the inclusion of open source tools, DRiVE now scaled to deliver a product for anyone interested in their next move.

Lunch

  1. 12:20pm-1:30pm

    Lunch + OpenSource GIS Doctor's Office

    A Bunch of CUGOSians, Makers of Geosptaial Things

    Stop by the Doctor's Office while gorging on pizza. This is perfect place to get answers to the questions you've been burning to ask. How do i setup Git, Qgis, PostGIS on my laptop so i can follow along in the HowTo sessions? What is the best OpenSource tool for my specific problem? What types of Desktop OpenSource GIS tools are there? How do I do everything with open-source tools that i once did with ArcGIS tools? I want to download OSM data, edit it and push changes back -- what tech do I use? Is Leaflet or OpenLayers a better client-side library? Show me how to write the TileMill carto styling language! What is MapBox and how can i make a webmap in 15 minutes...let's do it!

HowTo Sessions

A Session Reminder

These are not workshops!
The purpose of this section is to have an experienced presenter walk through a workflow, software demo or cookbook. We encourage people to follow along closely on their laptops. There will be CUGOS helpers on standby to assist with downloads, installs and general tech hurdles ( at least we think ). But we will not be making sure everyone is on the same page. Presenters will have a very detailed and public instruction manual so people following along can walk through it later on their own ( again, we think this will probably happen ).

  1. 1:30pm-3:00pm

    Learn How to Edit OpenStreetMap to Add Your Favorite Hangout

    Clifford Snow and Chase Stephens, OSM Seattle CEO(s) and General Badasses

    Get a jump on the Fall editathon on October 19th by learning how to use JOSM for some good-ole armchair mapping!

  2. 1:30pm-3:00pm

    The Python GDAL/OGR Cookbook Sessions

    Cort Daniel, Michael Payne, Greg Corradini

    Learn how to use the Python GDAL/OGR API with code snippets from this awesome cookbook. The web site is a project at GitHub and served by Github Pages. If you find missing recipes or mistakes in existing recipes please add an issue to the issue tracker.
